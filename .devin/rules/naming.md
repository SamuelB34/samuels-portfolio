---
description: Project naming conventions
---

- Components: PascalCase (MyComponent.tsx)
- Styles: kebab-case (my-component.module.scss)
- Functions: camelCase (scrollToElement)
- Constants: UPPER_SNAKE_CASE
- Files: kebab-case for utilities, PascalCase for components
- SCSS classes: BEM (block__element--modifier)
- CSS variables: --font-* for fonts, --*-rgb for colors
- TypeScript interfaces: PascalCase with descriptive names
- Props interfaces: always named `Props` or `Params`
